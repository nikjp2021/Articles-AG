# Articles-AG: The Automated AI Content Pipeline 🚀

Welcome to **Articles-AG**, a fully autonomous, multi-agent pipeline designed to research, write, enrich, and review high-quality business articles across multiple social media platforms.

This project solves the problem of scaling high-quality, platform-specific content creation. Instead of relying on manual drafting and review, Articles-AG uses a team of specialized AI agents to generate compelling narratives, source real citations, create accompanying visual assets, and enforce strict quality control before anything is published.

## What This Project Does

The pipeline takes a simple topic or prompt (e.g., *"The WMD Paradox"*) and automatically executes a **6-Step Flow**:

1. **Research**: Scours the web for trending data, verifiable statistics, and viral angles, outputting a structured Research Brief.
2. **Drafting**: Writes four distinct, platform-optimized articles for:
   - **LinkedIn**: Professional, analytical, and data-driven.
   - **Facebook**: Conversational, relatable, and community-focused.
   - **Threads**: Punchy, raw, and contrarian.
   - **Instagram**: Highly visual carousel formats with explicit slide breakdowns.
3. **Organization**: Automatically structures all outputs into dated, topic-specific directories for easy tracking.
4. **Visual Enrichment**: Generates highly specific AI image prompts and physical image assets tailored to each platform's aesthetic.
5. **Quality Review**: A dedicated reviewer agent scores the articles against a strict **13-Point Quality Rubric**.
6. **Auto-Correction**: If any article fails the review, the pipeline automatically logs the mistake and rewrites the content until it passes.

## The 13-Point Quality Standard

Every piece of content generated by this project must pass a rigorous 13-criteria review, ensuring human-level quality:
1. **Hook strength**: Must instantly grab attention.
2. **Citation quality**: Mandatory verifiable references (no vague "recent research").
3. **Virality potential**: Built-in engagement triggers.
4. **Platform tone fit**: Proper voice for the specific network.
5. **Tone differentiation**: Distinct angles across all 4 platforms.
6. **Emoji strategy**: Appropriate and balanced visual breaks.
7. **Structure & flow**: Easy readability and logical progression.
8. **Length appropriateness**: Optimized for platform algorithms.
9. **Visual/formatting**: Clean markdown, bolding, and line breaks.
10. **Context grounding**: Real-world relevance and stakes.
11. **Title format**: Strict `**Title:**` block requirements.
12. **Alt Text Awareness**: Mandatory accessibility descriptions for visual slides.
13. **No generic filler**: High signal-to-noise ratio.

## Project Structure

- `art-DD-MM-YYYY/` – Contains the final outputs, including the Research Briefs, the 4 platform articles, the generated `Assets/` images, and the strict `review.md` scorecards. Check out the sample topics (like `Global-South` and `Space-Exploration`) to see the pipeline in action.
- `/.antigravity/` – Contains the underlying Antigravity AI agents and SKILL files that power the automation.
